# Windows Utilities

## Purpose
Utilities for Windows maintenance tasks.

| Tool | Language(s) | Description |
| :--- | :--- | :--- |
| [Delete Directory](./delete_directory.md) | Batch | Purges directories with deeply nested or long paths by mirroring an empty temp folder before removal. |
| [Delete Old System Directory](./delete_old_system_directory.md) | Batch | Takes ownership of `Windows.old` (or similar) and removes it after an OS upgrade. |
| [Delete MS Teams App Data](./delete_ms_teams_app_data.md) | Batch | Clears the Microsoft Teams cache by reusing the Delete Directory tool. |
